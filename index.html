Overview

Stylus is a training-free music style transfer framework that repurposes a pretrained image diffusion model (Stable Diffusion v1.5) to operate on mel-spectrograms. By interpreting mel-spectrograms as structured time–frequency images, Stylus leverages visual spatial priors to disentangle musical structure (geometry) from timbre (texture) — entirely without task-specific training or fine-tuning.

Novel Contributions

  • Training-free music style transfer via DDIM inversion on mel-spectrograms — no fine-tuning, no text prompts, no task-specific model required.
  • Repurposed image diffusion prior (Stable Diffusion v1.5) that treats mel-spectrograms as time–frequency images, exploiting visual spatial priors to separate structure from texture.
  • CFG-inspired dual guidance with query-preservation weight γ (content structure retention) and guidance scale α (style amplification), enabling fine-grained independent control over content and style fidelity.
  • Phase-preserving ISTFT reconstruction that reuses the content phase, eliminating the metallic artifacts common in Griffin-Lim iterative estimation.
  • Multi-style interpolation by linearly blending key/value attention features of two style references with a single scalar β, enabling smooth timbral transitions without any additional training.

Motivation

  1. Coarse text-based conditioning is insufficient. Existing methods such as MusicGen rely on text descriptions to specify style, but text cannot faithfully capture the fine-grained timbral nuances of a specific audio exemplar (e.g., the exact resonance of a particular erhu recording).
  2. Task-specific training is costly and inflexible. Textual inversion–based methods (e.g., MusicTI) require per-style optimization, making them computationally expensive and unable to generalize across new styles without re-training.
  3. Powerful image diffusion priors are underexplored for audio. Large-scale image diffusion models encode rich visual structure-texture disentanglement. Mel-spectrograms share analogous structure (time–frequency layout ↔ geometry) and texture (timbre ↔ appearance), making image diffusion models a natural, untapped prior for audio style transfer.

Style keys and values are extracted from the style path and injected into the content denoising path using an attention-based feature swap (StyleID). A CFG-inspired guidance term with scale α amplifies the style direction while a query-preservation weight γ retains the melodic structure of the content. The generated mel-spectrogram is converted back to a waveform using phase-preserving ISTFT reconstruction, which reuses the content phase to eliminate metallic artifacts common in iterative phase estimation.

Method

Stylus repurposes a pretrained image diffusion model (Stable Diffusion v1.5) for music style transfer by treating mel-spectrograms as structured time–frequency images. The key insight is that visual spatial priors naturally disentangle musical structure (analogous to image geometry) from timbre (analogous to texture).

The pipeline consists of three stages:

  1. DDIM Inversion. Both the content and style mel-spectrograms are inverted into the diffusion latent space without any training, yielding content latent zc and style latent zs.
  2. Attention-based Style Injection. During denoising, the self-attention keys and values from the style path are substituted into the content path (StyleID). The query from the content path is preserved with weight γ to anchor the melodic layout, while the guidance scale α amplifies the stylistic displacement in feature space (CFG-inspired guidance).
  3. Phase-Preserving Reconstruction. The denoised mel-spectrogram magnitude is combined with the original content phase and converted back to a waveform via ISTFT, avoiding the metallic artifacts of Griffin-Lim iterative estimation.

Multi-Style Interpolation

Stylus extends naturally to multi-style interpolation by linearly blending the key/value features of two style references. Given style A and style B, the mixed style features are computed as KVmix = (1 − β) · KVA + β · KVB, where β ∈ [0, 1] controls the contribution of each style. No retraining or fine-tuning is required — the interpolation operates entirely in the attention feature space of the frozen diffusion backbone. As β increases from 0 to 1, the output timbre smoothly transitions from Style A to Style B while the melodic content of the source track is preserved throughout.

Results

We evaluate Stylus on a benchmark of 13 content tracks × 18 style instruments (13,246 stylized samples total), comparing against MusicGen-melody and MusicTI.

  • User study (2,925 ratings, p < 0.05): Stylus achieves 34.1% higher content preservation and 25.7% better perceptual quality than the best baseline.
  • Quantitative metrics: Stylus outperforms baselines on FAD, KLD, and Chroma similarity, confirming improvements in both audio fidelity and style-content alignment.
  • Efficiency: Being training-free, Stylus requires no GPU-hours for per-style optimization, with inference time comparable to a single diffusion sampling pass (~50 DDIM steps).
  • Multi-style interpolation: The β-sweep across five values (0.1, 0.3, 0.5, 0.7, 0.9) shows smooth, monotonic timbral transitions while style fit metrics for each endpoint style change in the expected direction.
